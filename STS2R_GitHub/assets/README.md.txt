md # Assets This folder contains a reduced public subset of the visual assets used in the STS2R synthetic data-generation pipeline, including logo-like decals, material textures, and decorative interference patterns. The full internal asset library is not included in this GitHub repository because of its large size. The files provided here are intended only for lightweight reproduction, folder-structure reference, and example runs. If a full asset package is released separately, it should be downloaded and placed into this `assets/` directory following the same folder structure. Users may also replace the provided files with their own compatible assets
